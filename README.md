## 🌟 The Vision

**FateLife** is an immersive, completely client-side, text-based life simulation engine. Seamlessly blending the rich, sandbox-style decision matrix of classic simulation with elegant, high-contrast modern web interfaces, FateLife invites players to forge their own paths through billions of life-path variations.

From your very first breath to your golden years, every tap of the **Age** button advances your personal narrative. Will you become a high-earning brain surgeon, a real-estate mogul, a loving parent, or an infamous mastermind running from the law? Your destiny is entirely in your hands.

### 1. The Core Attribute Wheel
Your avatar's success is governed by a network of six dynamic, interconnected vital statistics:
*   **😊 Happiness:** Dictates mental wellness, event resiliency, and relationship success.
*   **💪 Health:** Your physical life-force. Sinks under stress or disease; rises with active gym routines and physician interventions.
*   **🧠 Smarts:** Determines academic eligibility, scholarships, high-level career entry, and general puzzle resolution.
*   **✨ Looks:** Affects dating market opportunities, public engagement success, and unique interactions.
*   **☯️ Karma:** Elevated by selfless, supportive decisions; hidden weight affecting random event distributions and survival odds.
*   **⚡ Stress:** Tracks academic, corporate, or financial strain. Leaving stress unmanaged leads directly to severe physical and mental burnout.

## 🛠️ Technical Design & Craftsmanship

FateLife represents a masterclass in modern, client-side, zero-database state management.

```
                  ┌────────────────────────────────────────┐
                  │          VITE DEVELOPMENT HOST         │
                  └───────────────────┬────────────────────┘
                                      ▼
                  ┌────────────────────────────────────────┐
                  │          ZUSTAND STORE ENGINE          │
                  │   (Maintains entire state-tree: HUD,   │
                  │   Finances, Assets, Crime, Relations)  │
                  └──────┬──────────────────────────┬──────┘
                         │                          │
                         ▼                          ▼
         ┌──────────────────────────────┐  ┌──────────────────────────────┐
         │       GAME LOOP TICKER       │  │      DETERMINISTIC RNG       │
         │   • Annual Age Increments    │  │   • Condition-based Events   │
         │   • Balance-sheet calculations│  │   • Stats Degradation        │
         │   • Compound Interest drift  │  │   • Random Disease Spikes    │
         └──────────────────────────────┘  └──────────────────────────────┘
```

### 📦 Key Architectural Pillars:
1.  **Unified State Store (Zustand):** Leverages a robust, reactive Zustand state slice architecture (`src/gameStore.ts`). Every financial transaction, attribute degradation, age advancement, and family interaction resolves through deterministic state-mutating actions with flawless atomic performance.
2.  **Modular React Architecture:** Avoids monolith code pitfalls. Distinct UI tabs—including the *CareersPanel*, *AssetsPanel*, *PrisonYardPanel*, *RelationshipsPanel*, and *PetShoppingMarketplace*—are compartmentalized into clean, decoupled files, preventing generation truncation and maximizing codebase agility.
3.  **Tailwind CSS Polish:** Features a high-contrast dark visual theme inspired by premium slate dashboards. Elegant animations from `motion/react` provide satisfying, responsive transitions for game events, trial modals, and annual timeline additions.
4.  **Deterministic Event Registry:** Hundreds of rule-bound narrative occurrences (`src/data/events/`) trigger procedurally based on your character's current age, finances, relationship status, criminal standing, and geographic parameters.

## 🚀 Quick Start Guide

### 1. Prerequisites
Ensure you have **Node.js 18+** installed on your workstation.

### 2. Installation
Clone the repository and install all required framework packages:
```bash
npm install
```

### 3. Spin up Dev Server
Launch the superfast local Vite dev server on the default container port:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) inside your web browser to start playing instantly.

### 4. Build for Production
Bundle the optimized client-side single page application (SPA):
```bash
npm run build
```
Static static files are rendered securely to the `/dist` directory.
